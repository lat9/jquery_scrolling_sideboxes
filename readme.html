jQuery Scrolling Sideboxes — for Zen Cart v1.5.5 and later

Version 2.1.2, by lat9

Current Support Thread at Zen Cart Forums: http://www.zen-cart.com/showthread.php?214591

What it does

This plugin provides four (4) additional sideboxes to your store's configuration:

  1. sideboxes/best_sellers_scrolling.php
  2. sideboxes/featured_scrolling.php
  3. sideboxes/specials_scrolling.php
  4. sideboxes/whats_new_scrolling.php

In each case, the sidebox is heavily modelled after the built-in Zen Cart sidebox of a similar name with one difference: each sidebox uses some jQuery to show one product at a time in the sidebox, cutting down on the screen "real-estate" required.

Note: Starting with v2.0.0 of the plugin, your active template must load a version of jQuery for the plugin to properly operate!

Installation

This plugin currently has no core-file overwrites; you should always backup your cart's database and files prior to making any changes.

  1. Unzip the plugin's package file, which you've most likely already done if you're reading this!
  2. Copy the contents of the /includes/templates/YOUR_TEMPLATE/jscript directory to the like-named subdirectory for your store's active template:
    1. /includes/templates/YOUR_TEMPLATE/jscript/jscript_sidebox_slider.js
  3. Copy the plugin's new, default files to your store's file system.
    1. /includes/languages/english/extra_definitions/scrolling_sideboxes.php
    2. /includes/modules/sideboxes/best_sellers_scrolling.php
    3. /includes/modules/sideboxes/featured_scrolling.php
    4. /includes/modules/sideboxes/specials_scrolling.php
    5. /includes/modules/sideboxes/whats_new_scrolling.php
    6. /includes/templates/template_default/jscript/jscript_sidebox_slider.js
    7. /includes/templates/template_default/sideboxes/tpl_best_sellers_scrolling.php
    8. /includes/templates/template_default/sideboxes/tpl_featured_scrolling.php
    9. /includes/templates/template_default/sideboxes/tpl_specials_scrolling.php
    10. /includes/templates/template_default/sideboxes/tpl_whats_new_scrolling.php
  4. Log into your Zen Cart admin and use the Tools->Layout Boxes Controller to enable and position the sideboxes for your store's use.
